Álgebra

Su objetivo es hallar el valor de un variable

Que es una variable, algo desconocido o que varia

            3 + 2 - 24 = x
                |                   |
            Ecuación             Variable

Una ecuación es un conjunto de números, operadores y algo desconocido separado por un igual

El "=" representa la igualdad entre dos expresiones que contienen una o más variables

Reglas Básicas

            1 + 2 = x   3 + 2 = x
               x = 3      x = 5

Sistemas de Ecuaciones

  • Una variable siempre es la misma

                            12 / 3 = x   4 + 2 = x
                        x + x = 4    3 + 1 = x    2 + 2 = x

  • Dos variables pueden ser iguales

                            x + y = 4
                        2 + 2 = 4    3 + 1 = 4

Propiedades de las ecuaciones

              a = b        b = c          c = a
            1 + 2 = 3    3 = 4 - 1    1 + 2 = 4 - 1

Si añadimos aun lado un elemento, debemos hacerlo al otro lado tambien

                  a = b       a + x = b + x       a - x = b - 3
                1 + 2 = 3   1 + 2 + 2 = 3 + 2   1 + 2 - 1 = 3 - 1

Ejemplo

                    x + 3 = 5 + 2
                x + 3 - 3 = 5 + 2 - 3
